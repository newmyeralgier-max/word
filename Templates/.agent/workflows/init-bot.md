# Создание нового Бота (Scraper / Telegram)

Используйте этот воркфлоу, когда хотите начать новый проект с ботом. Я автоматически создам структуру папок и базовый код, устойчивый к ошибкам.

1. Запроси у пользователя название для нового бота/папки.
2. Создай проектную директорию с этим названием.
3. Внутри директории создай файл `.env` с шаблонными переменными (например, `TELEGRAM_TOKEN=`, `PROXY_URL=`).

// turbo

4. Создай виртуальное окружение: `python -m venv venv` и установи базовые библиотеки: `.\venv\Scripts\activate; pip install requests beautifulsoup4 python-dotenv selenium loguru`.

5. Создай файл `main.py` со следующей базовой структурой:
    - Настройка логгера (`loguru`).
    - Загрузка `dotenv`.
    - Блок `try-catch` в главной функции (self-healing паттерн).
    - Базовая структура для работы с прокси и headless режимом (если это веб-скрапер).
6. Выведи пользователю сообщение об успешном создании проекта и предложи заполнить файл `.env`.
