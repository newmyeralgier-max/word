# Линтинг и Форматирование (Python)

Этот воркфлоу запускает популярные инструменты форматирования (Black, isort) и проверки (Ruff/Flake8), чтобы привести код в соответствие со стандартами PEP8.

// turbo-all

1. Проверь наличие виртуального окружения `venv`. 
2. Выполни установку линтеров: `.\venv\Scripts\activate; pip install black isort ruff`.
3. Запусти isort для сортировки импортов: `.\venv\Scripts\activate; isort .`
4. Запусти black для форматирования кода: `.\venv\Scripts\activate; black .`
5. Запусти ruff для поиска логических ошибок: `.\venv\Scripts\activate; ruff check . --fix`
6. Выведи отчет о том, какие файлы были изменены и остались ли предупреждения от Ruff.
