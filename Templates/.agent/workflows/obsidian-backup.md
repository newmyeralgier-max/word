# Бекап базы Obsidian (Git)

Этот воркфлоу автоматически индексирует новые заметки в вашем хранилище Obsidian и пушит их в репозиторий для резервного копирования. Идеально для конца рабочего дня.

// turbo-all

1. Перейди в корневую папку хранилища Obsidian.
2. Выполни `git status`, чтобы проверить, есть ли изменения.
3. Выполни `git add .` для добавления всех новых заметок и вложений.
4. Выполни получение текущей даты: сгенерируй коммит с сообщением в формате "backup: obsidian vault backup [текущая дата и время]".
5. Выполни `git commit -m "..."`.
6. Выполни пуш в репозиторий: `git push`.
7. Выведи сообщение об успешном резервном копировании базы знаний.
