# Запуск Бэкенда (FastAPI)

Этот воркфлоу автоматически устанавливает зависимости и запускает сервер разработки. Благодаря флагу `// turbo-all`, я выполню все эти команды в фоне без дополнительных вопросов!

// turbo-all

1. Перейди в директорию `converter_app/backend` (используя `run_command` с нужным `cwd`).
2. Проверь наличие виртуального окружения `venv`. Если его нет, создай командой `python -m venv venv`.
3. Активируй виртуальное окружение и установи зависимости: `.\venv\Scripts\activate; pip install -r requirements.txt`.
4. Запусти сервер `uvicorn` в фоновом режиме: `.\venv\Scripts\activate; uvicorn main:app --reload`. Оставь команду работать (используй `WaitMsBeforeAsync`).
5. Прочитай логи сервера через `command_status`, чтобы убедиться, что он стартовал без ошибок (на порту 8000).
6. Открой `http://localhost:8000/docs` в моем встроенном браузере (`read_browser_page`), чтобы проверить доступность Swagger UI.
