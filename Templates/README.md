# Шаблон Базового Проекта

Это многоразовый шаблон для запуска новых проектов с поддержкой ИИ.

## Структура

- `.agent/` — Предварительно настроенные навыки (skills) и рабочие процессы (workflows).
- `.vscode/` — Настройки среды и линтинга.
- `directives/` — Документация для Уровня 1 (Директивы).
  - `templates/` — Готовые шаблоны для новых функций и баг-фиксов.
- `execution/` — Python-скрипты для Уровня 3 (Исполнение).
  - `utils/` — Полезные утилиты: `collector.py` (сбор контекста проекта) и `logger.py` (логирование).
- `data/`, `tests/`, `prompts/` — Стандартные папки для данных, тестов и промптов.
- `.tmp/` — Временные файлы.
- `GEMINI.md` — Основные принципы архитектуры проекта.

## Стандарт Языка

**ОБЯЗАТЕЛЬНО**: Все инструкции, документация, СОПы и пояснения (включая `GEMINI.md`, `README.md` и проектную документацию) ДОЛЖНЫ быть написаны на **русском языке**.

## Начало Работы

1. Скопируйте эту папку в новое местоположение вашего проекта.
2. Создайте виртуальное окружение: `python -m venv .venv`
3. Активируйте его: `.venv\Scripts\activate` (Windows)
4. Установите базовые зависимости: `pip install -r requirements.txt`
5. Начните работу, следуя указаниям в `GEMINI.md`.
