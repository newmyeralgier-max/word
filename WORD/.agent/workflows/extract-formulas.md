# Извлечение формул в LaTeX

Этот воркфлоу сканирует документ, находит все формулы и конвертирует их в двойной формат (обычная запись + LaTeX-блок кода).

// turbo-all

1. Определи исходный файл — спроси у пользователя путь к `.docx` или `.pdf` файлу.
2. Если файл PDF — попроси пользователя сконвертировать в `.docx` (LaTeX из PDF видно лучше, чем из Word).
3. Запусти сканирование формул:
   ```powershell
   & "C:\Users\New-life\AppData\Local\Programs\Python\Python313\python.exe" execution/latex_converter.py --scan "путь/к/файлу.docx"
   ```
4. Проанализируй найденные формулы:
   - Проверь корректность LaTeX-нотации
   - Убедись что формулы совпадают с оригиналом (не искажены)
   - Проверь что все греческие буквы и индексы корректно конвертированы
5. Покажи результат пользователю в формате:
   ```
   Формула (текст): r = 15 · tg(45°) = 15 м
   
   LaTeX:
   ```latex
   r = 15 \cdot \tan 45^\circ = 15 \,\text{м}  
   ```
6. Для одиночной формулы:
   ```powershell
   & "C:\Users\New-life\AppData\Local\Programs\Python\Python313\python.exe" execution/latex_converter.py --formula "I = S / (√3 · U)" --fix-decimals
   ```
7. Сообщи пользователю: количество найденных формул, формат для вставки в Office, рекомендации.
