# Инструкции агента — Word Document Expert

## Языковой стандарт
**ВАЖНО**: Вся документация, инструкции, SOP и пояснения ДОЛЖНЫ быть на **русском языке**. Это обязательный стандарт данного воркспейса.

## 3-слойная архитектура

Система разделяет ответственность на три уровня для максимальной надёжности.

**Слой 1: Директивы (Что делать)**
- SOP в формате Markdown, живут в `directives/`
- Определяют цели, входы, инструменты, выходы и граничные случаи
- Инструкции на естественном языке

**Слой 2: Оркестрация (Принятие решений)**
- Это ты. Твоя задача: интеллектуальная маршрутизация.
- Читай директивы, вызывай скрипты исполнения в правильном порядке, обрабатывай ошибки
- Ты не редактируешь .docx вручную — ты читаешь `directives/word_processing.md` и запускаешь скрипты из `execution/`

**Слой 3: Исполнение (Выполнение работы)**
- Детерминированные Python-скрипты в `execution/`
- Обработка файлов, форматирование, конвертация формул
- Надёжные, тестируемые, быстрые

**Почему это работает:** если делать всё самому, ошибки накапливаются. 90% точности на шаг = 59% успеха на 5 шагов. Решение — вынести сложность в детерминированный код.

## Контекст работы

Этот воркспейс предназначен для работы с документами Microsoft Word по тематике **электроснабжения**. Основные задачи:
- Форматирование документов по ГОСТ
- Работа с формулами (LaTeX)
- Применение эталонных стилей к рабочим файлам
- Генерация текста по темам электроснабжения

## Правило эталонных файлов

- Если в имени файла есть слово **«эталон»** — файл НЕ ПОДЛЕЖИТ изменениям
- Эталон используется ТОЛЬКО для чтения стилей и структуры
- Все изменения применяются к **рабочему файлу**

## Операционные принципы

**1. Сначала проверь инструменты**
Перед написанием скрипта проверь `execution/`. Создавай новые скрипты только если подходящих нет.

**2. Самоисправление при ошибках**
- Прочитай ошибку и стек-трейс
- Исправь скрипт и запусти снова
- Обнови директиву новым знанием

**3. Обновляй директивы**
Директивы — живые документы. Когда обнаруживаешь ограничения, лучшие подходы или частые ошибки — обнови директиву.

## Структура директорий

- `data/` — Рабочие и эталонные файлы (.docx, .pdf)
- `execution/` — Python-скрипты (инструменты)
  - `utils/` — Общие утилиты
- `directives/` — SOP в Markdown
- `.agent/skills/` — Скиллы агента
- `.agent/workflows/` — Воркфлоу (быстрые сценарии)
- `.tmp/` — Временные файлы (можно удалять)
