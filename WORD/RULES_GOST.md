# Основные требования ГОСТ 7.32-2017 и 2.105-2019

Этот документ содержит ключевые выжимки из ГОСТов для быстрого доступа при разработке алгоритмов сборки и форматирования Word-документов.

## 1. Параметры страницы и текст
- **Поля:** левое — 30 мм, правое — 15 мм, верхнее и нижнее — 20 мм.
- **Шрифт:** Times New Roman, цвет — черный, размер — 14 пт (допускается 12 пт).
- **Межстрочный интервал:** полуторный (1,5).
- **Абзацный отступ:** 1,25 см.
- **Выравнивание:** по ширине.
- **Полужирный шрифт:** допускается только для выделения заголовков (при настройке под локальные вузовские методички часто отключается).
- **Висячие строки:** обязателен запрет висячих строк (widow/orphan control).

## 2. Заголовки
- **Структурные элементы** (ВВЕДЕНИЕ, ЗАКЛЮЧЕНИЕ, СПИСОК ЛИТЕРАТУРЫ): пишутся прописными (заглавными) буквами, посередине строки.
- **Разделы (H1):** Начинаются с новой страницы. Номер раздела ставится перед заголовком без точки. По текущей локальной методичке: выравнивание по центру, отступ 0, полностью НЕЖИРНЫЙ шрифт. 
- **Подразделы (H2, H3):** По текущей методичке выравнивание по центру, отступ 0, полностью НЕЖИРНЫЙ шрифт.
- **Специфика:** Точка в конце любого заголовка НЕ СТАВИТСЯ. Переносы слов в заголовках не допускаются.

## 3. Иллюстрации (Рисунки)
- **Размещение:** По центру страницы.
- **Подпись:** Размещается под рисунком, по центру. 
- **Формат подписи:** `Рисунок 1.1 — Название рисунка`. В конце подписи точка не ставится.

## 4. Таблицы
- **Название:** Размещается над таблицей, выравнивание слева без абзацного отступа. 
- **Формат названия:** `Таблица 1.1 — Название`.
- **Шрифт в таблице:** Допускается уменьшение размера шрифта (обычно 12 пт) и одинарный межстрочный интервал.
- **Автоподбор:** Ширина таблиц не должна выходить за поля документа (примерно 16,5 см).

## 5. Формулы
- **Размещение:** Выравнивание по центру.
- **Нумерация:** Обязательна, размещается по правому краю в круглых скобках на одном уровне с формулой: `(1.1)`.
- **Расшифровка:** Начинается со слова «где» без абзацного отступа, далее список переменных с их размерностями.

## 6. Списки
- Используются маркированные списки с символом тире («–») или строчные русские буквы со скобкой (а), б), в)).
- **Отступ:** Начинаются с абзацного отступа (1,25 см).
