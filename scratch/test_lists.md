# Тест списков

Оформление надписей:– Font — выбор шрифта;– Text alignment — выравнивание текста;– Flip name — перемещение подписи блока;– Show/Hide name — отображение или скрытие подписи.

Цветовое оформление:– Foreground color — цвет контура блоков;– Background color — цвет заливки блоков;– Screen color — фон рабочей области модели.

Расположение и отображение:– Flip block — зеркальное отражение блока;– Rotate block — поворот на 90° по часовой стрелке;– Show drop shadow — отображение тени;– Show port labels — отображение меток портов.
